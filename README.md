# Project B - CSV Mini Database and Query Engine

## Overview

Project B is a lightweight database engine designed to work with CSV (Comma-Separated Values) files. It loads CSV data into memory-efficient data structures and executes SQL-like queries against it. This project demonstrates fundamental database concepts including data storage, query parsing, and query optimization.

## Features

- **CSV File Loading**: Parse and load CSV files with configurable delimiters and header support
- **In-Memory Storage**: Efficiently store CSV data in memory-optimized structures
- **Query Language**: Execute SQL-like queries supporting:
  - SELECT with column projection
  - WHERE clauses for filtering
  - ORDER BY for sorting
  - JOIN operations between tables
  - Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- **Query Optimization**: Implement efficient query execution strategies
- **Error Handling**: Validate and report errors comprehensively

## Architecture

The project is organized into modular components:

- **CSV Parser**: Reads and parses CSV files with flexible formatting options
- **Table Storage**: Manages in-memory table structures with support for indexing
- **Query Engine**: Parses query strings and executes them against stored data
- **Result Handler**: Formats and returns query results to users
